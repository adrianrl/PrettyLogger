## Installation
This framework is compatible with Carthage or Cocoapods, use one of both.
### Carthage
Add this into your `Cartfile`:
```ogdl
github "sebastianvarela/PrettyLogger" 
```
### Cocoapods
Add this into your `Podfile`:
```ogdl
pod 'PrettyLogger' 
```
